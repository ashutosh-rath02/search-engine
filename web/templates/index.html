Search Techniques Explained

Basic Search

Keyword-based search with TF-IDF ranking. Matches exact words in documents.

Fuzzy Search

Handles typos and misspellings using edit distance. Try misspelling words!

Phonetic Search

Matches words that sound similar (e.g., "phone" and "fone").

Semantic Search

Understands meaning using word embeddings. Finds related concepts.

Combined Search

Uses all techniques above and combines results for best performance.
